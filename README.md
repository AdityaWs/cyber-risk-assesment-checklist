# Cyber Risk Assessment Checklist
PT Phillip Broker Asuransi Indonesia (PBAI) — Kuesioner Penilaian Risiko Cyber / Large Corp

*Repository untuk hosting aplikasi Claude (AI)*

Ini adalah Cyber Risk Assessment Checklist yang dibuat untuk PT Phillip Broker Asuransi Indonesia (PBAI).
Secara singkat, ini adalah aplikasi web interaktif untuk menilai risiko keamanan siber suatu perusahaan klien dari sudut pandang underwriting asuransi. Fungsinya:
Apa yang dilakukan:

* Menyajikan 138 pertanyaan yang dibagi ke 15 sub-kategori keamanan informasi (kontrol akses, backup, manajemen insiden, kepatuhan, dll.)
* Pengguna menjawab setiap pertanyaan dengan Ya / Tidak / N/A
* Sistem menghitung skor risiko otomatis (0–100) berdasarkan bobot tiap kategori

*Output yang dihasilkan:*
* Risk score keseluruhan beserta levelnya: Rendah / Sedang / Tinggi / Sangat Tinggi
* Breakdown skor per sub-kategori
* Rekomendasi untuk underwriter (apakah layak diasuransikan, perlu sub-limit, atau ditolak)
* Bisa dicetak atau diexport ke JSON

*Siapa yang menggunakannya:*
Kemungkinan digunakan oleh broker atau underwriter asuransi saat melakukan due diligence terhadap calon nasabah yang ingin membeli asuransi cyber.
Intinya, ini alat bantu untuk mengukur seberapa "aman" sistem IT sebuah perusahaan sebelum diputuskan apakah mereka layak mendapat perlindungan asuransi siber dan dengan syarat seperti apa.
